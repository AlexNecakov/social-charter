    Validation & Error Handling:
        On failed network requests, show a user-friendly error message (not just console errors).

# How to run locally:
1. `git clone <to wherever>`
2. `npm install`
3. (If auth is needed)
  - add auth token to `.env.local`
4. `npm run dev`
5. go to `http://localhost:3000`
